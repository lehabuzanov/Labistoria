# Краткая инструкция пользователя

## 1. Импорт списков

- Откройте вкладку `Библиотека`.
- Нажмите `Импортировать XML из папки проекта`, если исходные XML уже лежат в репозитории.
- Для новых списков используйте `Загрузить новые списки`.

## 2. Создание выравнивания

- Перейдите на вкладку `Выравнивание`.
- Выберите главный список.
- Укажите листы главного списка.
- Отметьте остальные списки и укажите соответствующие листы.
- Нажмите `Выполнить автоматическое выравнивание`.

## 3. Просмотр корпуса

- Выберите нужное сохранённое выравнивание.
- Настройте порядок следования списков.
- Ограничьте отображаемые списки и типы разночтений.
- При необходимости сузьте диапазон строк.

## 4. Ручная правка

- Укажите номер строки.
- Выберите список, в котором нужно поправить выравнивание.
- Используйте:
  - `Сдвинуть вверх`
  - `Сдвинуть вниз`
  - `Вставить пустую выше`
  - `Вставить пустую ниже`
  - `Склеить с нижней`
  - `Удалить пустую строку`
- Если автоматическая классификация ошиблась, вручную измените тип разночтения.

## 5. Сохранение и продолжение работы

- Отдельная кнопка “сохранить проект” не нужна: все правки сохраняются в постоянное хранилище.
- После повторного открытия приложения откройте нужное сохранённое выравнивание и продолжайте.

## 6. Экспорт

- В блоке `Экспорт XML-TEI` нажмите `Сформировать XML-TEI для параллельного корпуса`.
- Затем скачайте полученный файл через кнопку `Скачать XML-TEI`.
